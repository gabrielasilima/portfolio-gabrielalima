Projeto JobDay

O Jobday é um projeto de UI/UX design desenvolvido no Figma, focado na criação de um layout moderno e funcional para um aplicativo mobile voltado ao mercado de trabalho. A proposta do aplicativo é atuar como um intermediador entre empresas que desejam divulgar vagas de emprego e candidatos em busca de novas oportunidades. O layout foi planejado para proporcionar uma navegação simples e objetiva, com telas intuitivas tanto para recrutadores quanto para candidatos.
